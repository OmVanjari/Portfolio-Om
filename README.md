# Om Vanjari | Professional Portfolio

A modern, high-performance, and fully responsive portfolio website designed to showcase my journey as a Fullstack Developer. Built with a focus on clean aesthetics, smooth interactions, and accessibility.

![Portfolio Preview](profile.jpeg)

## 🚀 Features

- **Dynamic Theme System**: Seamless switching between Light and Dark modes with persistent user preference using LocalStorage.
- **Glassmorphic UI**: Modern interface using backdrop filters, subtle gradients, and standard Bootstrap components.
- **Responsive Design**: Optimized for everything from small mobile screens to large desktop monitors.
- **Interactive Elements**:
  - Scroll-triggered animations for a dynamic feel.
  - Smooth navigation with Bootstrap's ScrollSpy integration.
  - Hover effects and micro-interactions on cards and buttons.
- **Contact Integration**: Fully functional contact form powered by Formspree.
- **Clean Architecture**: Modular CSS using variables for easy theme customization.

## 🛠️ Built With

- **Structure**: [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
- **Styling**: [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) & [Bootstrap 5.3](https://getbootstrap.com/)
- **Icons**: [Font Awesome 6.4](https://fontawesome.com/)
- **Typography**: [Google Fonts](https://fonts.google.com/) (Inter & Poppins)
- **Interactions**: Vanilla [JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- **Deployment**: [Render](https://render.com/) (Recommended) / GitHub Pages / Vercel

## ⚙️ Setup & Customization

1. **Clone the repository**:
   ```bash
   git clone https://github.com/1jariom/portfolio.git
   ```

2. **Open the project**:
   Simply open `index.html` in any modern web browser.

3. **Update Theme Colors**:
   Go to `style.css` and modify the `:root` variables to change the primary accent colors.

4. **Change Form Destination**:
   Update the `action` attribute in the `<form>` tag within `index.html` to your own Formspree endpoint.
